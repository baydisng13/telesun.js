## рѕўрїЇрЅбрІФ

рЅ┤рѕїрѕ░ріЋ рЅдрЅх рІерѕџрѕѕрІІрІѕрїЦ рІерЇЋрѕ«рїЇрѕФрѕЮ ріЦріЊ рІеріаріерЅБрЅб рІхрѕе-рїѕрїЙрЅй рІерѕџріЊрїѕрѕерІЇ ріарЅхріГрѕЇрЅх ріљрІЇ рЅ░рїЇрЅБрѕФрІі рѕўрѕерїЃрІјрЅх ріЦріЊ рЅ░рїарЅЃрѕџ рѕўрІ░рЅЦ рІерѕЮріЋріќрѕГ ріЦрІЇріљрЅ░ріЏ ріљрІЇрЇб

рЅ┤рѕїрѕ░ріЋ ріе JavaScript ріЦріЊ [Apps Script](https://developers.google.com/apps-script) рЅарѕџрїѕріЮ ріарІГріљрЅх рЅдрЅх рѕІрІГ рІерѕєріљрІЇріЋ ріЦрЅхрѕЮ рѕўрїарЅђрѕЮ ріарІхрѕГрїІрЅйріІрѕЇрЇб

### ріарѕЏрѕФрї«рЅй

- рЅа [Telegram Bot API 6.0](https://core.telegram.org/bots/api) рІерЅ░рїарЅђрѕЎ ріЦрЅхрѕЮ
- рЅарЅ░рѕѕрІФрІЕ рѕўрѕерїЃрІјрЅх рѕўріерЅ│рЅ░рѕЇ рІФрѕѕ
- рЅа Google ріЦріЊрЅХріЋ ріЦрЅхрѕЮ рѕІрІГ рЅа `Youtube`, `Drive`, `Gmail...` рЅ░рЅЦрѕЈрѕЇ
- рЅарїірІю рЅарЅ░рїерѕЏрѕф рІѕрїЦрЅ░ріЋ рІѕрІ░ Google Cloud рѕўрїѕріўріЋ
- рЅарѕФрѕ▒ рЅа рІе Google рЅ░рѕЏрѕф (Google sheet) рѕѕрѕўрїарЅђрѕЮ ріарѕЇрЅ╗рѕЇріЕрѕЮ
- рІерѕЮріЋрїѕріЮ рѕхрѕѕ рЅарѕѕрїа рІе `100+` рЅдрЅх рѕўрѕерїЃ рІГрїарЅђрѕЎ
- рЅарѕўріЋрїѕрІх рІѕрІ░ рЅ░ріерЅ░рѕѕ рІе `Basic Javascript` ріЦріЊ ріЦріЋрІ░рѕџрїарЅђрѕЎ рѕўрЅђрѕўрїФ ріљрІЇ

### рѕЮріЋрІЏрѕг

рЅарѕўріерЇІрЅ▒ ріерЅ░рѕўріљрѕ╗ `рѕўрѕерїЃ ріЦріЋрІ░рѕџрѕ░рїарІЇріЋ рЅаріарЇЋрѕх рѕхріГрѕфрЇЋрЅх` ріЦріЊ рЅарѕїрѕІ ріГрЇЇрѕЇ рІЇрїцрЅ▒ ріерѕЮрѕ│рѕї рѕўріерЇІрЅх ріЦріЊ `рѕЮрѕІрѕй рЅ░рїарЅЁрѕўріЋ рІѕрІ░ рЅ░рѕЏрѕфрІјрЅй рѕІрІГ рѕўрѕўрїѕрЅЦ` ріарѕЇрЅ╗рѕЇріЕрѕЮ рЇб

```js
// ріеріарЇЋрѕх рѕхріГрѕфрЇЋрЅх рІЇрѕхрїЦ рѕўрѕўрїѕрЅЦ ріерѕЏріЋрЅарѕерЅх рЅарЅ░рѕўріљрѕ╗ рІерЅ░рѕўрІ░рЅа рІерЇІрІГріЊ рЅхріГріГрѕѕріЏ ріаріФрЅБрЅб рІГрѕ░рїарІІрѕЇ
function WelcomeToTelesun() {
  // рѕѕрѕўрѕўрїѕрЅЦ рІерЅ░рЅђрЅарѕѕ рІерЅ│ріљрЅа ріарЇЋрѕх рѕхріГрѕфрЇЋрЅх рІЇрѕхрїЦ рЅ░ріерЅ│рЅ│рІГ рІерѕєріљ рѕхріЕрІЋ рЅХріГріЋ рІГрїарЅђрѕЎ
  const bot = new Bot.Telesun(<<botToken>>);

  // рѕѕрѕўрЅєрїБрїарѕГ рѕѕріаріЋрІ░ріЏрІЇ рІерѕхріљрѕЇрЅдріЊ рѕ░рІЇ рѕЏрѕЁрЅарѕерѕ░рЅЦ рІГріќрѕГрЅарЅ│рѕЇ
  bot.Use((ctx) => ctx.reply("рѕ░рѕІрѕЮ рѕ░рѕІрѕЮ!"));

  // рѕѕрѕўріерЅ│рЅ░рѕЇ <</start>> рі«рѕЏріЋрІх рѕІрІГ рІерЅ░рЅђрЅарѕѕ рѕхріЕрІЋ рЅХріГріЋ рІГрїарЅђрѕЎ
  bot.Start((ctx)=> ctx.reply("рІГрѕЁ рѕЮріЋрІЏрѕг рі«рѕЏріЋрІх ріљрІЇ"))

  // рѕѕрѕЮрѕ│рѕї <hello> рѕўрѕЇрІЋріГрЅх рЅ░рїарЅђрѕўрЅарЅх рІерЅ░рЅђрЅарѕѕ рѕхріЕрІЋ рЅХріГріЋ рІГрїарЅђрѕЎ
  bot.Hears('hello', (ctx)=> ctx.reply("рІГрѕЁ рІерѕўрѕЇрІЋріГрЅх рѕўрѕЇрІЋріГрЅх ріљрІЇ"))

  // рѕѕрѕЮрѕ│рѕї рѕјріГ рЅ░рѕўріЋ рѕІрІГ рІерЅ░рѕўріљрѕ╗ рѕўрѕЇрІЋріГрЅх рЅ░рїарЅђрѕўрЅарЅх
  bot.Photo((ctx)=> ctx.reply("рІГрѕЁ рѕјріГ ріљрІЇ"))
}
```

рѕѕрЅ░рїерѕЏрѕф рѕўрѕерїЃ рѕЮріЋрІЏрѕг [рѕЮрІЮрїѕрЅБрІЇ](https://github.com/abdiu34567/telesun.js/wiki/Example-Bots) рІЮрѕГрІЮрѕЕріЋ рІГрѕўрѕЇріерЅ▒
